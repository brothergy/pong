# pong
A sandbox to learn love2d using the `pong` game.

## Install love2d
https://love2d.org/

## Download this repo


## Run love

```r
> love ~/path/to/mygame
```

## New features
Added: 
- Random color changing ball 
- Forward/backward movement to advance onto opponents side
